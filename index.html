For this exercise I specifically investigated the use cases for web components in HTML where we create custom elements with custom logic and display them using plain vanilla Javascript instead of using a frame work.

Each of these "folder like" containers are 2 custom elements, a "file-folder" and a "file-element", and the folders that contain more than 1 page, will have buttons added automatically to them, to flip, as though we're reading a book. If you squint, it will at least look like that :P.

Each custom file-element is also scrollable! So the user defines the height on the file-folder element and if the individual file-elements require more space, they just get a scrollbar. Neat!
